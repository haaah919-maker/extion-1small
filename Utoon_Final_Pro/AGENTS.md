# إرشادات للمطورين (Utoon Reader Pro Max v3.0)

هذه النسخة تم بناؤها لتكون الأقوى والأكثر أماناً لموقع utoon.net.

## المميزات التقنية:
1. **سحب الصور عبر API:** تم استخدام `wp-json` الخاص بـ Madara Theme لجلب الصور مباشرة، مما يحل مشاكل الفصول المقفولة أو الأسماء التي تحتوي على نقاط (مثل 25.2).
2. **التحميل المتعدد:** تم إضافة أزرار PDF و ZIP مباشرة بجانب كل فصل في قائمة الفصول، مع زر لتحميل الكل.
3. **نظام العضويات (Supabase):**
   - يتم إنشاء معرف فريد لكل مستخدم تلقائياً.
   - يتم تتبع الاستهلاك اليومي (فصل واحد للمجاني).
   - تفعيل الـ VIP أو Lifetime يتم بتغيير قيمة `vip_until` في قاعدة البيانات.
4. **الإعلانات:** تظهر الإعلانات تلقائياً للمستخدمين المجانيين فقط عند فتح القارئ.

## كيفية تفعيل VIP لمستخدم:
اذهب إلى لوحة تحكم Supabase، جدول `users`، وقم بتعديل حقل `vip_until` للمستخدم المطلوب:
- للـ VIP لمدة شهر: ضع التاريخ القادم بصيغة Timestamp.
- للـ Lifetime: ضع قيمة كبيرة جداً (مثلاً `4102444800`).

## هيكلة الملفات:
- `supabase_client.js`: المسؤول عن التواصل مع قاعدة البيانات.
- `background.js`: يدير التثبيت، الحدود اليومية، وحقن السكربتات.
- `content_script.js`: يضيف الأزرار لواجهة الموقع الأصلية.
- `reader_logic.js`: محرك القارئ المخصص والتحميل داخل الفصل.
