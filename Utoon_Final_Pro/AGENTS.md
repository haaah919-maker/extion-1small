# Utoon Ultimate Pro Max v1.6 (Official Documentation)

هذه هي النسخة النهائية والكاملة (The Ultimate Version) التي تجمع كل المميزات من النسخ السابقة مع تحسينات جذرية في الأداء والشكل.

## المميزات الجديدة (v1.6):
1. **إزالة شكل الماوس المزعج:** تم استعادة الماوس الأصلي للمتصفح مع إضافة تأثيرات بصرية احترافية عند التحويم فوق الأزرار فقط.
2. **تأثيرات بصرية Ultra Legendary:**
   - **Legendary Magic:** كرات سحرية مع تلاشي وضبابية.
   - **Inferno Storm:** تأثيرات نارية سريعة وحركية.
   - **Nebula Stars:** نجوم ثابتة تتلاشى وتظهر في خلفية المجرة.
   - **Midas Touch:** سقوط ذهبي ملكي مع دوران سريع.
3. **محرك سحب الصور الذكي:**
   - يحاول أولاً عبر API للحصول على أعلى جودة.
   - يقوم بالتخمين الذكي (Brute Force) للفصول المقفولة.
   - يسحب من الصفحة مباشرة كخيار أخير.
4. **تحسين UI/UX:** أزرار بتدرجات لونية (Gradients) وتأثيرات زجاجية (Glassmorphism).

## إدارة النظام:
- **Daily Limit:** فصل واحد يومياً للزوار، يتم تتبعه عبر Supabase.
- **Bypass:** النسخة تتخطى حماية الموقع الأصلية وتعرض المحتوى مباشرة في واجهة سينمائية.

## ملفات هامة:
- `content_script.js`: المحرك الرئيسي للقارئ والتأثيرات.
- `background.js`: إدارة الهوية والاستهلاك.
- `bulk_helper.js`: معالج تحميل الـ PDF و ZIP.
